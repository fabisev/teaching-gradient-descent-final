# References and Publication Notes

## Core content sources for this book

1. Bishop, C. M. (2006). *Pattern Recognition and Machine Learning*. Springer.
   This book informed the treatment of the gradient descent update rule, the distinction between batch and stochastic updates, and the discussion of optimization in linear models and neural networks.

2. Muller, A. C., and Guido, S. (2016). *Introduction to Machine Learning with Python*. O'Reilly.
   This book informed the practical framing around learned weights, neural-network structure, model complexity, feature scaling, and convergence behavior in applied machine learning.

3. Menon, A. (2018, September 16). [Linear Regression using Gradient Descent](https://medium.com/data-science/linear-regression-using-gradient-descent-97a6c8700931). *TDS Archive / Medium*.
   This article was used as the source for the valley analogy figure.

4. Olamendy, J. C. (2025, August 26). [Gradient Descent in Deep Learning: A Complete Guide with PyTorch and Keras Examples](https://medium.com/@juanc.olamendy/gradient-descent-in-deep-learning-a-complete-guide-with-pytorch-and-keras-examples-e2127a7d072a). *Medium*.
   This article informed the practical framing around neural-network training, learning-rate problems, stochastic updates, and how frameworks automate gradient computation.

5. GeeksforGeeks. (2026, May 12). [Gradient Descent Algorithm in Machine Learning](https://www.geeksforgeeks.org/machine-learning/gradient-descent-algorithm-and-its-variants/).
   This resource informed the concise treatment of batch, stochastic, and mini-batch gradient descent and the simple multi-parameter update example.

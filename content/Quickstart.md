# Reading Path

This page is kept for compatibility with the target book scaffold. The active gradient-descent book uses the following reading path:

1. Start with the [overview](00-overview/00-overview.md) to understand the learning goals and experimental comparison.
2. Read [Part I: Classic Textbook and Formula](10-classic-textbook/10-classic-textbook.md) for the traditional instructional version.
3. Read [Part II: Multiple Modalities](20-multiple-modalities/20-multiple-modalities.md) for geometric, physical, worked-example, and interactive explanations.
4. Use the [common questions](06-transfer-and-exercises/06-transfer-and-exercises.md) after either instructional condition.

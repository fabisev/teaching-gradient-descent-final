# Introduction

Gradient descent is a core machine learning concept, but it can be difficult for beginners because it connects definitions, symbolic notation, and step-by-step reasoning. This book supports a teaching experiment that compares two ways of presenting the same topic.

The content is aligned around the same learning goals in both instructional conditions. Learners should be able to explain why gradient descent is used, define loss, gradient, and learning rate, interpret the update rule step by step, predict the effect of different learning rates, and apply the reasoning to a new example.

The active book structure is defined in `toc.yml`; this scaffold page is retained only so the target project layout remains intact.

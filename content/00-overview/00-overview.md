# Overview

Gradient descent is a core machine learning concept, but it can be difficult for beginners because it connects definitions, symbolic notation, and step-by-step reasoning. This book is structured to support a teaching experiment that compares two ways of presenting the same topic.

## Background

The purpose of the experiment is to compare whether students understand gradient descent differently depending on how the material is presented.

The concept stays the same in both conditions. What changes is the instructional format.

## How the experiment works

This book is organized into two instructional conditions:

- **Part I: Classic Textbook and Formula**
- **Part II: Multiple Modalities**

Both parts cover the same beginner-level topic and are meant to address the same content scope. The difference is that the first part presents the material in a more traditional text-and-formula style, while the second part presents the same concept using several modalities.

After the learning material, there is one separate chapter with **common questions for both conditions**. These questions are intended to be used with either instructional version.

Items about confidence, clarity, cognitive load, or learner opinion are **not** included in the questions chapter. Those belong in a separate survey.

## Shared learning goals

Both instructional conditions are designed around the same learning goals. After either version, students should be able to:

- explain why gradient descent is used to improve model parameters
- define **parameter**, **loss**, **gradient**, and **learning rate**
- interpret the update rule step by step
- predict what happens when the learning rate is too small, reasonable, or too large
- distinguish batch, stochastic, and mini-batch gradient descent at a beginner level
- apply gradient descent reasoning to a simple new example

The two parts therefore teach the same topics. The classic part uses a textbook sequence of definitions, formulas, and short explanations. The multimodal part uses pictures, analogy, a worked example, and an interactive activity to support the same ideas.

## Book structure

1. **Part I: Classic Textbook and Formula** presents the traditional instructional version.
2. **Part II: Multiple Modalities** presents the multimodal instructional version.
3. **Common Questions for Both Conditions** contains the shared questions that can be used after either instructional condition.
